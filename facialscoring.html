REGA Facial Scoring Study

The standard scoring approach is using social and demographic data, but we are going to introduce new type of model - facial scoring, where score is calculated based on the member’s face photo. The model prototype is already developed and tested and gives about 70% accuracy. Now we are working to improve this figure. To build the model we are using Machine Learning and Cognitive Service Provided by Microsoft. The Microsoft Face API (MS Cognitive) was used for the member photo processing where around 15000 pictures were digitized. For each face picture a flag attribute (good/bad) is calculated based on traditional scoring model (base model). The base scoring model is real scoring model that is used by several banks to issue loans to the private individuals. There are the following face landmarks/attributes that we are using for the facial scoring model: age, gender, head pose, smile, facial hair and glasses.
